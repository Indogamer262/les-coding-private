## Palet warna

- **Biru tua**: `#111e3f` (sidebar navigation)
- **Putih**: `#ffffff` (background utama dan card)
- **Hijau mint/emerald**: `#047857` (ikon total pengajar)
- **Oranye hangat**: `#ea580c` (ikon pembelian / warning)
- **Ungu**: `#7c3aed` (ikon paket aktif)
- **Biru**: `#1e3a8a` (ikon total murid)

## Menjalankan di lokal

1. Buka terminal dan masuk ke folder ini.
2. Jalankan server PHP built-in:

	`php -S localhost:8000 -t .`

3. Buka di browser:

	`http://localhost:8000/`
